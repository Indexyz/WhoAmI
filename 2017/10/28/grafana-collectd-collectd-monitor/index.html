环境

本文接下来都在 Arch Linux 上安装 大部分软件的搭建使用 Docker 来保证别的系统也可用
数据储存在 /data

安装 Docker

对于其他系统, 官方已经准备了一键安装脚本 执行就好了

对于 Arch Linux 官方的源中已经有了 Docker 的存在

Pull 所需的映像

  • influxdb:alpine
  • grafana/grafana
  • busybox:latest

Collectd

对于 Collectd 我们使用从包管理器安装的方法安装 而不是运行 Docker 映像 因为 Collectd 的数据收集 CPU 内存 网络使用 需要收集宿主机的信息

默认包管理已经有现成的了

其他系统请参考 官方教程
